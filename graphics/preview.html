Hero Banner 1280 × 400 GitHub README · Social preview
Export to PNG:
Browser: open hero.svg directly, set DevTools viewport to 1280×400, screenshot.
CLI: rsvg-convert -w 1280 -h 400 hero.svg > hero.png
Or: npx svg-to-png hero.svg hero.png --width 1280 --height 400
